# Desafío Técnico — Sistema de Solicitudes Internas

## Requisitos previos

- PHP 8.2+
- Composer
- Node.js 18+

## Setup

```bash
git clone <este-repo> desafio-tecnico
cd desafio-tecnico
bash setup.sh
php artisan serve
```

Abrí el navegador en `http://localhost:8000`.
